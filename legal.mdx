## DMCA and the injection DLL

Rose **cannot distribute the injection DLL** due to DMCA restrictions. You must obtain this file yourself from an authorized source and sign it with your own code signing certificate.

On first launch, Rose detects that the DLL is missing and prompts you to place it in the correct folder. See [DLL setup](/configuration/dll-setup) for details.

## No competitive advantage

Rose strictly modifies **local rendering variables** — the models and textures your own client displays. It does not:

- Manipulate network packets or game server data
- Read or write game memory states
- Alter gameplay mechanics, hitboxes, or any stat
- Provide information unavailable to other players

Rose is designed to give zero competitive advantage. This is a core design constraint, not an afterthought.

## Analytics disclosure

Rose collects **anonymous, aggregate usage data** to help the maintainers understand how many unique users are active. Specifically:

- A periodic ping is sent every 5 minutes to the analytics server at `api.leagueunlocked.net`.
- The machine identifier is derived from the **Windows Machine GUID** — a hardware-level ID that contains no personal information.
- No usernames, IP addresses, summoner names, skin selections, or any personally identifiable information are collected or stored.
- Analytics run on a background thread and have no impact on performance.

You can disable analytics at any time in the Rose **Settings** panel.
